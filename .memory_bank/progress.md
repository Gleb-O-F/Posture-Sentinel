# Progress: Posture Sentinel

## Статус проекта: **Python версия (v0.1)**

## Режимы работы

### 1. Калибровка (Calibration)
- Активируется клавишей **'c'**
- Сохраняет baseline: плечи, уши, глаза
- Параметры сохраняются в config.yaml

### 2. Боевой (Monitoring)
- Мониторинг осанки в реальном времени
- Детекция нарушений с таймаутом 3 сек
- Уведомление текстом + планируется размытие

## Что готово
- Python версия с MediaPipe
- 33 ключевые точки (BlazePose)
- Отрисовка скелета
- Калибровка (клавиша 'c')
- Детекция: slouching, leaning_forward, head_down
- Счетчик FPS
- Сохранение config.yaml

## Что не готово (по PRD)
- ❌ Таймаут 3 секунды до срабатывания
- ❌ Forward Lean по расстоянию между ушами (15%)
- ❌ Наклон головы по углу (>15°)
- ❌ **Размытие экрана** (ключевая фича!)
- ❌ Сохранение ear_distance в baseline
