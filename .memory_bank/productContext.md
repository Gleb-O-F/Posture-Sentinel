# Product Context: Posture Sentinel

## Проблема пользователя
Длительная работа за компьютером приводит к нарушению осанки (сутулость, наклон головы), что вызывает:
- Боли в спине и шее
- Снижение продуктивности
- Долгосрочные проблемы со здоровьем

## Зачем нужен проект
Автоматический мониторинг осанки в фоновом режиме с мягкими уведомлениями (размытие экрана), когда пользователь начинает сутулиться.

## Пользовательский опыт
1. Запуск приложения → захват камеры → отображение видео со скелетом
2. Пользователь нажимает 'c' для калибровки правильной позы
3. Приложение мониторит осанку в реальном времени
4. При нарушении >3 секунд → постепенное размытие экрана
5. Возврат в правильную позу → восстановление четкости

## Конкурентные преимущества
- Локальная обработка (приватность)
- GPU-ускорение
- Энергоэффективность
- Мягкая система уведомлений (не раздражает)
