# Active Context: Posture Sentinel

## Текущий фокус
**Инициализация проекта** — создание базовой структуры Rust-приложения.

## Активные решения
- Язык: Rust (стабильный)
- CV: opencv 0.92
- ML: ort 2.0.0-rc.11 с CUDA
- UI: winit 0.29
- Конфиг: toml + serde

## Модули (заготовки)
- `src/main.rs` — точка входа
- `src/camera.rs` — захват видео с веб-камеры
- `src/inference.rs` — инференс ONNX модели
- `src/posture.rs` — анализ осанки
- `src/config.rs` — загрузка/сохранение конфигурации
- `src/overlay.rs` — визуализация скелета и эффекты

## Приоритеты
1. Реализация захвата камеры (camera.rs)
2. Загрузка и инференс ONNX модели (inference.rs)
3. Детекция ключевых точек (posture.rs)
4. Отрисовка скелета (overlay.rs)
5. Интеграция с winit для окна и эффектов

## Известные неопределенности

- Нужен ли эффект Acrylic/Blur через window-vibrancy?
- Конфигурация камеры (FPS, разрешение) — не определено
