# Active Context: Posture Sentinel

## Текущий фокус
Доработка детекции и подготовка к ключевой фиче - размытию экрана.

## Режимы работы

### Калибровка (клавиша 'c')
- Сохраняет текущую позу как baseline
- Записывает: shoulder_y, ear_y, eye_y, shoulder_width
- Минусы: не сохраняет ear_distance (расстояние между ушами)

### Боевой (мониторинг)
- Сравнивает текущие landmarks с baseline
- Проверяет: плечи, уши, глаза
- Проблема: срабатывает мгновенно, нет таймаута 3 сек

## Приоритеты
1. Добавить таймаут 3 секунды
2. Реализовать размытие экрана (Qt/ctypes или overlay)
3. Улучшить детекцию по PRD (forward lean 15%, угол >15°)

## Известные проблемы
- Нет таймаута - сразу показывает "BAD POSTURE"
- Размытие экрана сложно реализовать в Python (нужен Win32 API)
