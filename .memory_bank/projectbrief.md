# Project Brief: Posture Sentinel

## Обзор
AI-приложение для мониторинга осанки в реальном времени на Rust.

## Целевая платформа
- Windows 10/11
- GPU: NVIDIA Quadro P1000 (или аналогичный с CUDA)

## Ключевые цели
| Цель | Описание |
|------|----------|
| **Производительность** | GPU инференс (CUDA), целевой FPS: 20-30 |
| **Энергоэффективность** | GPU ≤15%, CPU ≤5% в фоновом режиме |
| **Приватность** | Локальная обработка без облака |

## Основные функции
1. Детекция 33 ключевых точек тела (BlazePose)
2. Мониторинг осанки (сутулость, наклон плечей, наклон головы)
3. Калибровка базового положения
4. Визуализация скелета
5. Система уведомлений (размытие экрана)

## Модели
- BlazePose (MediaPipe) в формате `.onnx`
- Файлы: `pose_estimation_mediapipe.onnx`, `pose_landmarks_detector_full.onnx`, `pose_detection.onnx`
